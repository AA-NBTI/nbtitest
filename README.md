# NBTI Next.js Solution

당신의 성향을 정확하게 측정하는 차세대 트렌드 분석 시스템, **[NBTI테스트](https://nbtitest.com)** 프로젝트입니다.

## 🚀 프로젝트 소개
본 프로젝트는 기존 MBTI의 정형화된 틀을 넘어, 사용자의 행동 데이터를 기반으로 입체적인 성향을 분석하는 **NBTI(Next Behavioral Trend Indicator)**의 공식 웹 서비스입니다.

### 🎯 핵심 키워드 및 서비스
- **[NBTI테스트 공식 홈페이지](https://nbtitest.com)**: 실시간 성향 분석 및 트렌드 리포트 제공
- **트렌드 분석 엔진**: 쇼핑, 연애, 직장 등 다양한 상황별 성향 측정
- **광고 인프라**: 사용자 맞춤형 브랜드 매칭 및 광고 인벤토리 구조화

## 🛠️ 기술 스택
- **Framework**: Next.js 14+ (App Router)
- **Database/Backend**: Supabase (Auth, DB, Storage)
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Graphics**: Chart.js / Framer Motion
